Obsidian Sync Lab

このミニサイトでは、メモアプリ Obsidian を題材に、 PARA(P) メソッドによるディレクトリ設計、Remote Sync プラグインと S3 バケットを使った自己ホスト型同期、 さらに Ubuntu 環境でのセットアップまでをまとめて紹介します。

01 — Structure

PARP メソッドで組む Obsidian ディレクトリ

Projects / Areas / Resources / Archive をベースにした情報整理術 PARA（ここでは PARP）メソッドを、Obsidian のフォルダ構成に落とし込みます。 実際のディレクトリ例と運用イメージ付き。

02 — Sync

Remote Sync プラグイン + S3 同期

Obsidian 公式 Sync を使わずに、 Remote Sync（Remotely Save 派生プラグイン）と Amazon S3 互換のストレージを使って 自前で同期環境を構築する流れを整理します。 端末ごとの設定のポイントも。

03 — Ubuntu

Ubuntu に Obsidian を入れて同期する

Ubuntu 上に Obsidian をインストールする方法（Snap や AppImage）と、 Remote Sync + S3 を使って他の端末と同じボルトを同期する手順をまとめます。
